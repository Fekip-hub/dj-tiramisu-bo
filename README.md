# DJ Tiramisu - Bot de música para Discord

## Variables de entorno necesarias en Railway

- `DISCORD_TOKEN`: el token de tu bot (lo obtienes en https://discord.com/developers/applications)

## Comandos disponibles

- `/play <busqueda>` — reproduce una canción (nombre o URL de YouTube)
- `/skip` — salta la canción actual
- `/pause` — pausa la música
- `/resume` — reanuda la música
- `/stop` — detiene todo y limpia la cola
- `/queue` — muestra la cola de canciones
- `/leave` — desconecta el bot del canal de voz
